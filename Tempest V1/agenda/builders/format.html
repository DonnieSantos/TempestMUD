Colors And File Formatting.

Eventually, we will have an online system for builders to use, thus sparing them the tedious process of constructing their zones with the actual files. Until such a system is ready, we want to provide our builders with enough technical information so that they can understand how to build the zone themselves.

Please not that constructing these actual files is completely optional. Given the basic information for your zone, we could easily construct the files for you. If you do not feel that you understand the format of the files, please keep your building work abstract.

There are three file examples available here online, for a Zone File, Mobile File, and Item File. They are pretty self-explanatory to the experienced mudder. There are a few technical details that I want to make sure are completely spelled out, in case of any confusion. If you feel this section is missing anything, please let us know.

Zone Size And ID Numbers.

Technically, a zone can only be 100 rooms big. However, the code can merge more than one zone together and call it one zone. Odessa for instance is zone 15. There are 500 rooms allocated for it. That means the room numbers will range from 1500-1999. It's not such a big deal, since zone 15 still refers to all of those rooms, even the ones out of 1500-1599 range. The thing is, is that now there can be no zone 16, 17, 18, or 19. Because those room IDs are taken. All this means is that you should report early on if your zone is going to be very large, so that we can assign you more than 100 rooms.

Description Spacing.

There is a 500 character limit to each line that the file processor will read. To be on the safe side, make sure that any long descriptions have enough line-breaks in them so that you can read it all without having to scroll horizontally.

Also, never ever EVER use double spaces. Not after periods, nowhere. If a space is required to separate two words, always use one single space.

How To Use Colors.

To add color to a word, put a color mark before it, and after it to change it back.

          n = White      N = Grey (Normal)
   r = Red    R = Dark Red
   g = Green    G = Dark Green
   b = Blue    B = Dark Blue
   y = Yellow    Y = Dark Yellow
   c = Cyan    C = Dark Cyan
   m = Magenta    M = Purple

Usage:    A pound sign followed by the letter of the available color.
Example:    This is a #rRed#N Rose.
Output:    This is a Red Rose.
